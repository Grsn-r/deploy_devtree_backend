# 🌳 DevTree API

Backend de **DevTree**, una aplicación inspirada en LinkTree donde los usuarios pueden crear y administrar una página personalizada con enlaces a sus redes sociales.

Esta API proporciona autenticación, gestión de usuarios, perfiles, enlaces personalizados y carga de imágenes.

## 🚀 Tecnologías

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- bcrypt

## ✨ Funcionalidades

- Registro e inicio de sesión
- Autenticación mediante JWT
- Gestión de perfiles de usuario
- CRUD de enlaces sociales
- Reordenamiento de enlaces
- Subida de imágenes de perfil
- API REST para consumo desde el frontend
